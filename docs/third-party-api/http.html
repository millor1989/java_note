HTTP 请求

HttpClient

出现非常早的 Http 工具。

之前属于 Apache 的 Common HttpClient 项目。

目前属于 Apache HttpComponents 项目，包含在 HttpClient 和 HttpCore 模块中。

与 OKHttp 相比，稍显落后和臃肿。

OKHttp

3.0 版本之后称为 OKHttp3。

在 Android 项目开发中使用非常广泛。

4.0 之后的版使用 Kotlin 语言编写。

Unirest

轻量级；十分简洁、底层使用 HttpClient。
