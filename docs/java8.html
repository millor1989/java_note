Java 8

Java 8

1、新特性

  • Lambda 表达式
  • 函数式接口
  • 方法引用与构造器引用
  • Stream API
  • 借口的默认方法与静态方法
  • 新的日期时间 API
  • 其它

其中，引用最广泛的是 Lambda 表达式和 Stream API。

2、优点

  • 速度更快
  • 代码更简洁（新增语法 Lambda 表达式）
  • 强大的 Stream API
  • 便于并行
  • 最大化减少空指针异常 Optional
